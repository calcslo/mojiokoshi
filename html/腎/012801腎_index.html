試験に出る箇所（重要トピック）

・成人ネフローゼ診断基準：1日の蛋白尿3.5 g/日以上、血清Alb ≤ 3.0 g/dLが必須（「1,2が必須条件であり」PDF p.5）。理由：ネフローゼの基本定義であり国家試験・臨床試験で頻出の根拠的基準であるため。

・Selectivity Index (SI)：SI＝(尿中IgG/血中IgG)÷(尿中トランスフェリン/血中トランスフェリン)、SI≦0.2は高選択性（微小変化群→ステロイド感受性）（PDF p.6「SI≦0.2 高選択性」）。理由：蛋白尿の選択性は治療反応予測（ステロイド感受性）に直結し出題頻度が高い。

・PLA2R抗体（膜性腎症）：特発性膜性腎症の主要抗原はM型ホスホリパーゼA2受容体(PLA2R)で、血清PLA2R抗体は患者の活動性指標となる（PDF p.11「特発性膜性腎症の主たる原因抗原は...PLA2R」）。理由：MNの病因マーカーとして近年の教科書的知識であり臨床診断で頻出。

・MNの電顕ステージ：Stage I～IVの経過（上皮下電子密度沈着→spike形成→基底膜内埋没→電子密度低下）（PDF p.17～p.22）。理由：MNの病理診断で必須の分類であり病期と治療判断に関連するため出題されやすい。

・MCNSの治療反応：微小変化型は90%以上がステロイドで寛解（PDF p.44～p.54）。理由：小児・成人での治療反応性は国家試験の定番知識であり臨床的優先度が高い。

・MPGNと低補体：MPGNは低補体（特にC3低下）を示すことが多い（PDF p.29, p.35）。理由：補体低下を問う問題は鑑別診断で頻出であるため。

・FSGSの臨床的特徴：FSGSは巣状・分節性硬化、しばしばステロイド抵抗性で進行性に腎不全に至る（PDF p.72）。理由：治療抵抗性・予後に関する出題が多く、病理像と臨床像の関連が試験で問われやすい。

・ABPMと家庭血圧の重要性（講師）：家庭血圧の早朝測定（起床後1時間以内）・就寝前測定、原則2回測定・差があれば3回目を推奨（講師発言）。理由：診断精度を左右する実務的指導であり二次性高血圧の初期評価に頻出するため。

・原発性アルドステロン症のスクリーニング所見：アルドステロン過剰ではレニン低下・アルドステロン高値の組合せが示唆される（講師発言）。理由：二次性高血圧の代表的疾患でありスクリーニング・確定法が試験でよく問われる。

・ネフローゼ合併症：血栓・脂質異常管理：ネフローゼでは脂質異常や血栓リスク増加のためスタチンや抗凝固を考慮する（PDF治療スライド）。理由：合併症管理は臨床試験での出題頻度が高い基礎知識である。

試験に出ない箇所

・個別患者の診察室血圧159の具体例：講師の具体症例説明（"診察室では血圧が159となっていて..."）は臨床文脈の雑話であり、国家試験の汎用知識ではないため（講師発言）。

・講師の現場運用上の雑談（援助を要請する発言等）：講義中の運営上の発言（"誰か、援助してもらっていいですか?"等）は教育的核心ではないため（講師発言）。

・個別の薬剤が"高齢で下がりすぎる"という局所的経験談：講師の経験共有として述べられた高齢者での薬剤の過降圧例は具体的だが試験出題対象としては一般化しにくい（講師発言）。

・講師の院内手続き・推奨する外来運用の細かい指示：外来の受付対応やチーム編成に関する指示は教育上の業務連絡に近く国家試験の出題優先度は低い（講師発言）。

・講師の身体測定での椅子高さや細かい体位指示の個別例：心臓と同じ高さで測定する等の細部は重要だが試験で細かく問われるより臨床実務での注意点であるため優先度は低い（講師発言）。

・講師の一部の曖昧な推測・未確定コメント（"多分これ、正解でも2つが測定できていない場合もある"等）：確定的知識ではなく講師の臨床疑問提示であり試験問題化しにくい（講師発言）。

全体の詳細なまとめ

ネフローゼ症候群総論

ネフローゼ症候群は大量蛋白尿に伴う臨床症候群であり、代表的所見は大量蛋白尿、低アルブミン血症、脂質異常、浮腫である。講義では成人ネフローゼの診断基準として、1日蛋白尿3.5 g/日以上と血清Alb 3.0 g/dL以下(血清TP 6.0 g/dL以下)が必須で、脂質異常や浮腫は必須ではないと示された（PDF p.5「成人ネフローゼ症候群の診断基準」）。

尿蛋白の選択性

尿蛋白の選択性はSelectivity Index (SI)で評価され、定義は(尿中IgG/血中IgG)÷(尿中トランスフェリン/血中トランスフェリン)である。SI≦0.2は高選択性で主に微小変化群 (MCNS)を示唆しステロイド反応性が高い。SI>0.2は低選択性で微小変化以外（MN, FSGSなど）を示唆する（PDF p.6「尿蛋白の選択性」）。

膜性腎症（MN）

概要: 特発性膜性腎症は成人ネフローゼで頻度が高く、主因抗原はポドサイトのM型ホスホリパーゼA2受容体(PLA2R)であり、患者血清に対する抗体（PLA2R抗体）は臨床的マーカーとなる（PDF p.11「概要」）。日本人の約50%が陽性、抗体はIgG4が主体で、寛解前に消失し予後や活動性の指標となり得る。
病理: PAM染色でGBM肥厚と上皮側に突出するspikeを形成する（PAM所見, PDF p.13）。蛍光抗体では基底膜沿いにIgG, C3が顆粒状沈着。電子顕微鏡ではStage I～IVの経過が示され、Stage Iは上皮直下のelectron dense deposit、Stage IIではdeposit増加とspike形成、Stage IIIは基底膜内へ埋没、Stage IVは電子密度低下（PDF p.17～p.22）。治療は初期にプレドニゾロン(0.6～0.8 mg/kg/日)、抵抗例ではシクロスポリン(CyA)やシクロホスファミド併用を考慮する（PDF p.24～p.26）。

膜性増殖性糸球体腎炎（MPGN）

定義と特徴: メサンギウム増殖とメサンギウム間入（mesangial interposition）を来たしGBMの二重化を示す。一次・二次性があり、低補体（特にC3低下）が重要所見である（PDF p.29～p.34）。
病理: 光顕でびまん性増殖、PAMで不規則な基底膜二重化、蛍光でC3優位の沈着、電顕で内皮下の高電子密度沈着を認める。治療はステロイドや免疫抑制薬を要することが多く、予後は不良。

微小変化型ネフローゼ症候群（MCNS）

臨床: 小児に好発するが成人でも約38.7%を占める。発症は急激で浮腫が突然出現することが多い。尿蛋白は高選択性(SI<0.2)で、ステロイド反応性が良好（90%以上が寛解）である（PDF p.44～p.52）。
病理: 光顕で殆ど変化がないが、電顕で足突起融合（podocyte foot process effacement）を認める。治療はプレドニゾロン0.8～1.0 mg/kg/日（最大60 mg）で開始し、寛解後漸減、再発例や依存例では免疫抑制薬（シクロスポリン、ミゾリビン等）を追加する（PDF p.54～p.60）。

巣状分節性糸球体硬化症（FSGS）

概要と臨床像: MCNSと類似した発症様式を示すが、しばしばステロイド抵抗性で進行性に移行し末期腎不全に至ることがある。病理学的に部分的（focal）かつ分節的（segmental）な硬化を示す。遺伝性FSGSはNPHS2(podosin)やACTN4遺伝子変異が報告される（PDF p.72～p.78）。
治療: 初期はプレドニゾロン1.0 mg/kg/日を基本に、抵抗性例ではシクロスポリン、ミゾリビン、シクロホスファミド等を考慮する。補助療法としてACE阻害薬/ARBやLDLアフェレシス（難治例）など。

病理・検査のまとめ

- 光学顕微鏡、蛍光抗体法、電子顕微鏡（EM）は腎生検診断で必須であり、各疾患で典型所見が存在する（PDF各スライド、特にMN, MPGN, MCNSに関するEM/蛍光スライド）。
- 補体低下（CH50, C3, C4）はMPGNやSLEで重要（PDF p.35, p.36表）。

治療総論

- ネフローゼ型の免疫抑制戦略は病型（MCNS, MN, FSGS等）と重症度に依存する。ガイドラインに基づきステロイド単独または免疫抑制薬併用を選択する（PDF p.24, p.25, p.56等、エビデンスに基づくネフローゼ診療ガイドライン2014）。
- 補助療法として降圧薬（ACE阻害薬/ARB）、脂質管理（スタチン）、血栓予防（抗凝固/抗血小板）を行う。

臨床補助所見：尿沈渣・円柱

- ネフローゼでは脂肪円柱・卵円形脂肪体が出現することがあり、ろう様円柱は腎不全に関連すると国家試験的に教示されている（PDF p.40～p.42）。

講義: 高血圧と二次性高血圧（講師の追加講義）

講師の口述はネフローゼスライドのあとに高血圧・二次性高血圧の診療に移り、以下の点を強調した。
- 原発性アルドステロン症は二次性高血圧の重要な原因であり、アルドステロン過剰ではレニン低下・アルドステロン高値の組合せが示唆される。薬剤や電解質の影響で検査前に薬を中止する必要がある点を説明した。確定診断は複数検査での陽性や画像検査、副腎静脈サンプリングで局在診断を行う（講師発言）。
- 高血圧の評価で服薬遵守の確認、生活習慣（塩分・体重）把握が必須であると述べた。薬剤性（NSAIDs、ステロイド等）が血圧を上昇させること、腎機能や電解質異常が影響することを述べた。

血圧測定と家庭血圧（講師強調点）

- 測定方法の標準化が重要で、特に家庭血圧の朝測定（起床後1時間以内）を行い、可能なら就寝前も測定すること、各測定は原則2回行い差があれば3回目を測定することを推奨した（講師発言）。
- ABPM（24時間血圧測定）は診断精度向上に有用だが実施の難しさにも触れた。
- 診察室血圧だけに依存せず家庭血圧の記録を活用することで診断・治療方針が変わることを強調した。

原発性アルドステロン症の検査・治療フロー（講師）

- スクリーニングで異常を示した場合、薬剤調整後に確定検査、局在診断（CT/MRI、副腎静脈サンプリング）を行い、片側性であれば手術（副腎摘除）を検討、両側や非手術希望例では薬物療法で管理する（講師発言）。

生活習慣と治療アドヒアランス

- 塩分制限、減量、外食頻度の把握、患者教育が治療効果に直結するため臨床で重要であると繰り返し述べた。

睡眠時無呼吸症候群（SAS）

講義全体はネフローゼ症候群の病型別病理・診断（光顕・蛍光・電顕）と治療指針をPDFスライドに沿って整理し、加えて講師が臨床で強調した血圧測定の方法、二次性高血圧（特に原発性アルドステロン症）の診断フロー、服薬遵守・生活習慣の重要性を具体例を交えて述べた。PDFの主要スライド参照：成人ネフローゼ診断基準(PDF p.5)／尿蛋白選択性(PDF p.6)／MN概要・PLA2R(PDF p.11)／MN病理(EM, PAM)(PDF p.13～22)／MCNS治療(PDF p.54～60)／ガイドライン図(PDF p.24, p.56)。
