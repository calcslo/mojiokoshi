全体の詳細なまとめ

呼吸器感染症の概念と発症メカニズム

呼吸器感染症は全感染症の中でも頻度が高く、外界から吸入する空気に混在する病原体が気道に定着・増殖して炎症を起こすことで成立する。鼻・口から侵入し咽頭に定着すれば上気道感染（いわゆる風邪）となり、さらに下方へ落下侵入すると下気道感染（気管支炎）や肺炎に至る。
PDF p.2「病原体が外界から宿主へ侵入→上気道粘膜へ付着（結合）→…→下気道感染」

気道は鼻腔から咽頭・喉頭（耳鼻咽喉科領域）を経て、気管・気管支・細気管支・肺胞へ連続する。部位により病原体の親和性や炎症反応が異なり、鼻炎・気管支炎・肺炎など臓器別に起こりやすい病原体が異なる。
PDF p.3「気道と肺胞・間質…それぞれの組織により、原因微生物の種類や炎症反応、抗菌薬の濃度分布などが異なる」

気道感染症（気管支炎など）：急性と慢性、原因微生物の整理

下気道感染としての気管支炎は急性から慢性まで幅があるが、急性発症の多くはウイルス感染で、細菌は基本的に主因になりにくく、治療は対症療法が中心となる（気道が弱い人・基礎疾患がある人では細菌関与が増える）。
PDF p.4「急性：多くはウイルス、ときにグラム陽性球菌…基礎疾患ありでは…グラム陰性桿菌」

慢性・反復性（急性増悪を含む）では、ウイルスによる気道上皮障害を契機に細菌が定着しやすくなり、インフルエンザ菌、肺炎球菌が関与し、感染防御低下が進むと緑膿菌、クレブシエラなどが問題になりうる。
PDF p.6「慢性気道感染症（急性増悪）の原因菌…インフルエンザ菌、肺炎球菌、緑膿菌…」

急性気道感染の代表的ウイルスとして、成人の上気道炎はライノウイルス・コロナウイルス・アデノウイルス、インフルエンザは下気道にも影響しうる。小児は初感染・重症化しやすいウイルス（RSウイルス、パラインフルエンザなど）が下気道炎やクループの原因となる。
PDF p.5「急性気道感染症の病原ウイルス」

抗菌薬適正使用（急性気道感染症の基本方針）

急性気道感染症では不要な抗菌薬投与が耐性菌を選択しうるため、抗菌薬適正使用が重要である。急性気道感染症（普通感冒、急性鼻副鼻腔炎、急性咽頭炎、急性気管支炎など）に対して、抗菌薬が必要な場面を整理した手引きがある。
PDF p.7-8 厚労省「抗微生物薬適正使用の手引き 第一版」および診断手順

肺炎の疫学：高齢者肺炎の重要性

日本における肺炎死は高齢者に極端に偏り、肺炎死亡者の65歳以上が97.8%を占める。
PDF p.9「65歳以上 97.8%（2022年）」

入院・外来の肺炎患者数は高齢層で増加しており、肺炎は高齢化社会で大きな医療課題である。
PDF p.10「肺炎による入院および外来患者数の推移」

肺炎の鑑別：発熱＋肺陰影＝肺炎とは限らない

発熱や肺陰影の原因は感染症に限らず、心不全・肺水腫、肺癌（細気管支肺胞上皮型腺癌）、びまん性肺疾患（薬剤性肺炎、間質性肺炎、過敏性肺炎、膠原病肺、好酸球性肺炎、器質化肺炎、サルコイドーシス）、肺胞蛋白症、肺血栓塞栓症、放射線肺炎などを鑑別する必要がある。
PDF p.13「肺炎の鑑別」

肺炎の病理・画像パターン：肺胞性、間質性、混合性

肺炎の画像は病理パターンに対応して理解する。
・肺胞性肺炎（air space consolidation）：肺胞腔に滲出物が充満し白くなる（大葉性肺炎など）。
・間質性肺炎（ground-glass opacity）：炎症の主座が肺胞隔壁（間質）で、肺胞腔は比較的保たれ、薄い陰影（すりガラス）として見える。
・混合性肺炎：肺胞性と間質性の要素が混在（例：マイコプラズマ肺炎）。
PDF p.14「肺炎の病理組織パターン」
PDF p.15-20「肺炎の画像：肺胞性/間質性/混合性（肺炎球菌、レジオネラ、CMV、COVID-19、ニューモシスティス、マイコプラズマ）」

肺胞性陰影の特徴として、浸潤影の中で気管支が空気を含むため透亮に見える気管支透亮像（air bronchogram）が典型である。
PDF p.15「気管支透亮像（air bronchogram）」

間質性肺炎の例としてサイトメガロウイルス肺炎、COVID-19肺炎、ニューモシスティス肺炎などですりガラス陰影が示される。
PDF p.17-19

肺炎の分類：CAP/HAP/NHCAP と重症度・耐性菌

肺炎は発症環境・背景で病原体や耐性菌リスクが異なるため分類する。
・市中肺炎（CAP）：病院外の日常生活者に発症。
・院内肺炎（HAP）：入院後48時間以降に発症。
・医療・介護関連肺炎（NHCAP）：介護施設入所、90日以内退院、PS3以上の要介護者、透析など継続的血管内治療などの背景で発症。
PDF p.21-23「肺炎の定義・分類」

日本ではNHCAPが臨床的に多く、CAPとHAPの中間的性質（予後不良・耐性菌関与が増える）をもつ。
PDF p.23「CAP, NHCAP, HAPの位置付け」

死亡率はCAP＜NHCAP＜HAP＜VAPの順に高い。
PDF p.24「肺炎群別の死亡率（CAP 6.3%, NHCAP 15.5%, HAP 30.4%, VAP 32.4%）」

誤嚥性肺炎：高齢者での中心課題と予後

高齢化に伴い肺炎に占める誤嚥性肺炎の割合が増え、70–90歳代では過半数に達しうる。
PDF p.25「肺炎に占める誤嚥性肺炎の割合」

肺炎入院は体力低下→ADL低下→嚥下機能低下→再誤嚥→再肺炎→再入院という肺炎の負のスパイラルを形成し、予後を悪化させる。
PDF p.26「肺炎の負のスパイラル」

誤嚥リスク因子の保有は、短期（30日）死亡よりも、1年死亡や肺炎再発など長期予後に影響しうる。
PDF p.27「誤嚥危険因子保有は、長期予後や肺炎再発に影響」

肺炎入院後に認知機能低下が残存しうることが示され、肺炎は認知障害の進行にも関連する。
PDF p.28「肺炎による入院後の認知障害発症率」

成人肺炎診療ガイドライン2024：統合と基本戦略

従来、CAP/NHCAP/HAPでガイドラインが分かれていたが、臨床現場の要望もあり統合が進み、最新の成人肺炎診療ガイドライン2024では、敗血症性ショックの有無、重症度、耐性菌リスクを軸に治療場所と抗菌薬を決定する。
PDF p.29-31「ガイドラインの変遷」「フローチャート」

基本は、必要十分な検査で原因微生物同定（喀痰グラム染色・培養、抗原検査、核酸検査）を行い、初期はエンピリック治療、結果でde-escalation（狭域化）を行う。
PDF p.30-31

敗血症の評価：qSOFA/SOFA

感染症が疑われる場合、臓器障害を伴う敗血症を見逃さないため、qSOFAやSOFAで評価し、必要に応じて敗血症診療へ移行する。
PDF p.32「敗血症および敗血症性ショックの診断フローチャート」

CAP/NHCAPの重症度：A-DROP

CAP/NHCAPではA-DROPスコアで重症度評価を行う。
A: Age（男≥70、女≥75）
D: Dehydration（BUN≥21または脱水）
R: Respiration（SpO2≤90%またはPaO2≤60）
O: Orientation（意識変容）
P: Pressure（収縮期血圧≤90）
0項目=軽症、1–2=中等症、3=重症、4–5=超重症（敗血症性ショックがあれば1項目でも超重症）
PDF p.33

CAP：細菌性肺炎 vs マイコプラズマ肺炎の鑑別（日本独自の実用則）

CAPでは細菌性肺炎と非定型肺炎（主にマイコプラズマ）の鑑別が治療選択（βラクタム vs マクロライド/テトラサイクリン等）に直結する。鑑別項目は6つ：
1) 60歳未満
2) 基礎疾患なし/軽微
3) 頑固な咳嗽
4) 聴診所見乏しい
5) 迅速診断で原因菌証明されない（マイコプラズマ陽性を除く）
6) WBC <10,000/μL
5/6以上でマイコプラズマ強く疑う、3–4は鑑別困難、2以下で細菌性を疑う（高齢者で精度低下に注意）。
PDF p.34

マイコプラズマは飛沫感染で学校・家庭内の水平感染、流行性があり、重症例は気道狭窄を起こしうる。合併症として脳炎・髄膜炎、Guillain-Barré症候群など神経系合併症がある。
PDF p.35

マクロライド耐性マイコプラズマが増加しており、マクロライド無効例ではテトラサイクリン系/キノロン系への切替を考慮する（小児では副作用・適応の問題あり）。
PDF p.36

CAPのエンピリック治療（概要）

外来では、細菌性肺炎が疑われればアモキシシリン高用量など、非定型肺炎が疑われればミノサイクリンやマクロライド、またはレスピラトリーキノロンを選択する。入院ではSBT/ABPC、CTRX等を中心に、重症ではβラクタム＋マクロライドまたはキノロン併用などを検討する（レジオネラ疑いではキノロン/マクロライド）。
PDF p.37「CAPのエンピリック治療」

CAP原因菌として頻度が高いのは肺炎球菌、インフルエンザ菌、マイコプラズマなどである。
PDF p.38「市中肺炎(CAP)の原因菌」

個別菌種での薬剤選択の整理として、肺炎球菌はペニシリン系を基本に、必要に応じて第3世代セフェム、キノロンなどを位置づける。インフルエンザ菌ではβラクタマーゼ阻害薬配合ペニシリンやSBT/ABPC、BLNAR等を考慮する。
PDF p.39-40

レジオネラ肺炎：リスク、診断、治療

レジオネラ肺炎は水を介したエアロゾル感染で、市中では温泉・循環式浴槽・ビルのクーリングタワー、院内では給水系・浴槽/シャワー・ICUなどが関連する。尿中抗原（SG1）陽性率が近年60%程度に低下しており、陰性でも否定できない。軽症なら経口治療も可能で、土壌からのL. longbeachae感染（ガーデニング）も話題。
PDF p.41

レジオネラを疑う補助として診断予測スコア（男性、咳なし、呼吸困難、CRP≥18、Na≤134、LDH≥260：3点以上で疑う）が提示される。
PDF p.42

治療はキノロン（例：モキシフロキサシン、レボフロキサシン等）やマクロライドが中心となる。
PDF p.37,41

CAPのウイルス診断とMultiplex PCR

CAPでは細菌だけでなくウイルスが一定頻度で関与する（SARS-CoV-2流行前のデータ）。
PDF p.43「CAPにおけるウイルスの検出頻度」

Multiplex PCR（FilmArray®）は18種類のウイルス＋4種類の細菌を同時検出できるが、感度差、検体差、重複感染、定着の検出、細菌検出限界など解釈の注意点がある。
PDF p.44

NHCAP：耐性菌リスク評価とエンピリック治療

NHCAPはCAPと同様に敗血症評価とA-DROPで重症度を見つつ、追加で耐性菌リスクを評価して治療域を決める。
PDF p.30-31,46

耐性菌リスク因子として、挿管管理、過去90日以内抗菌薬使用、経腸栄養、低アルブミン血症、免疫抑制、過去90日以内入院、過去1年耐性菌検出歴などが挙げられる。
PDF p.46-47

NHCAPの検出菌は肺炎球菌・肺炎桿菌・MRSA・緑膿菌などが挙がるが、喀痰では定着の可能性が高く、解釈に注意する。
PDF p.48-50

NHCAPのエンピリック治療は、狭域（SBT/ABPCやCTRX±マクロライド、またはキノロン）と、耐性菌を考慮する広域（TAZ/PIPC、抗緑膿菌セフェム、カルバペネム等±抗MRSA薬）を耐性菌リスクに応じて選択し、培養結果でde-escalationを検討する。
PDF p.49

喀痰検査・グラム染色の位置づけ：Diagnostic Stewardship

喀出痰培養は原因菌を必ずしも意味しない。下気道局所痰の採取・定量培養が理想だが困難なことが多く、NHCAPでは耐性菌が分離されやすいが原因菌とは限らない。
PDF p.50

喀痰は上気道・口腔内常在菌で汚染されやすいため、Miller & Jones分類（肉眼評価）とGeckler分類（顕微鏡評価）で「原因微生物検索に適切な検体か」を判断する。
PDF p.51-54

誤嚥性肺炎：原因菌像と抗菌薬選択の考え方

誤嚥性肺炎では一般CAPより肺炎球菌が少なく、クレブシエラ属、MRSA、緑膿菌、嫌気性菌の割合が増える傾向がある。ただしMRSAや緑膿菌は定着が多く、初期からの広域抗菌薬（カルバペネム、TAZ/PIPC、抗緑膿菌キノロン等）使用はAMR対策上避けるべきとされる。
PDF p.55

嫌気性菌カバーの有用性はガイドラインのSRでは明確でなく、「必ずしも必須ではないが、どの症例で有益かは追加研究が必要」と整理されている。
PDF p.55

院内肺炎（HAP）と日和見感染

HAPは入院後48時間以降に新規発症した肺炎。背景は患者要因（基礎疾患、加齢、低栄養、誤嚥）、医原的要因（抗菌薬、抗がん剤、ステロイド、免疫抑制薬、生物製剤、ICI、デバイス：経鼻胃管、挿管、気管切開）、環境要因（アウトブレイク等）で、耐性菌感染・日和見感染・集団発生が問題となる。
PDF p.56

日和見感染症として、MRSA・緑膿菌等、真菌（カンジダ、アスペルギルス、ムーコル、ニューモシスティス肺炎）、ウイルス（サイトメガロウイルス、VZV）、抗酸菌（結核、NTM）などを念頭に置く。
PDF p.57

HAPの原因菌（網羅的解析）ではMRSA、緑膿菌、MSSA、肺炎桿菌、Enterobacter、Stenotrophomonas maltophilia、Acinetobacter、Serratia等が上位に挙がる。
PDF p.58

HAPの重症度（I-ROAD）と耐性菌リスク

HAPでは敗血症評価＋重症度評価（I-ROAD）＋耐性菌リスクで抗菌薬選択を決める。
I-ROAD：I（Immunodeficiency：悪性腫瘍/免疫不全）、R（Respiration：SpO2>90維持にFiO2>35%必要）、O（Orientation：意識低下）、A（Age：男≥70/女≥75）、D（Dehydration：乏尿/脱水）
これに加え、CRP≥20や陰影の広がり（一側肺2/3以上）などで群分け。
PDF p.60-61

耐性菌リスクはICU発症、敗血症/ショック、90日以内抗菌薬使用、活動性低下（PS≥3等）、CKDなどで、2個以上で高リスク。
PDF p.62-63

V-HAP/NV-HAP、VAP（人工呼吸器関連肺炎）

V-HAP（HAPから悪化して人工呼吸管理が必要）とNV-HAPで死亡率が異なり、V-HAPは高い。
PDF p.64

VAPは挿管・人工呼吸管理開始後48時間以後に新規発症した肺炎。発症率が高く、リスク因子は長期人工呼吸、再挿管、抗菌薬投与歴、基礎疾患、誤嚥、筋弛緩剤、カフ圧低下、移送、仰臥位、制酸薬など。
PDF p.65

挿管96時間以内の早期VAPは耐性菌可能性が小さく、96時間以降の晩期VAPはMRSA・緑膿菌・Acinetobacter等の耐性菌を考慮する。
PDF p.66

VAP治療はHAP原則に準拠し、培養後は速やかにde-escalation、感染否定なら中止、投与期間は7–8日を推奨。抗緑膿菌多剤療法は行わず、単独療法としてカルバペネム単剤を用いないなどの原則が示される。
PDF p.67

肺炎予防：手指衛生、うがい、マスク、誤嚥対策、口腔ケア

日本呼吸器学会の啓発「ストップ！肺炎」に沿い、予防の柱は日常対策とワクチンである。
PDF p.68,75-76

手洗いは上気道炎だけでなく肺炎予防にも有効。
PDF p.69

うがいは予防効果があり、水うがいで風邪罹患を40%低下させた報告がある。
PDF p.70

マスクは咳・くしゃみの飛沫拡散や吸入を抑える。
PDF p.71

誤嚥は顕性誤嚥（むせ等で明らか）と不顕性誤嚥（サイレントアスピレーション）があり、後者は睡眠中に口腔咽頭の細菌を知らずに誤嚥することが病態として重要。
PDF p.72

誤嚥性肺炎の防止は、摂食嚥下リハ、食事介助、食形態調整、嚥下訓練、口腔ケア、栄養ルート調整、頭位挙上、口腔内清拭、脱水・栄養改善、鎮静薬調整など多面的に行う。
PDF p.73

口腔ケアは誤嚥性肺炎発症率を下げる。
PDF p.74

ワクチン：肺炎球菌、インフルエンザ、COVID-19、RSV

予防の重点として、新型コロナワクチン、インフルエンザワクチン、肺炎球菌ワクチン、RSウイルスワクチンが挙げられる。
PDF p.76

●肺炎球菌
肺炎球菌はグラム陽性双球菌で莢膜をもち、血清型は約100種類。侵襲性肺炎球菌感染症（髄膜炎、敗血症、菌血症性肺炎など）の原因となり、莢膜がワクチン標的である。抗菌薬耐性も問題。
PDF p.77

肺炎球菌性肺炎のリスクは基礎疾患もあるが、年齢（高齢）が特に大きい。
PDF p.78

ガイドラインでは高齢者の肺炎予防に肺炎球菌ワクチンを推奨する。
PDF p.79

インフルエンザ罹患後の細菌性肺炎合併では肺炎球菌などが関与しうる。
PDF p.80

高齢者の肺炎予防にインフルエンザワクチン＋肺炎球菌ワクチン併用が推奨される。
PDF p.81

65歳以上成人の肺炎球菌ワクチンは、PPSV23、PCV15、PCV20、PCV21などがあり、推奨接種の考え方が整理されている。
PDF p.82

●インフルエンザ
インフルエンザウイルスはHA/NAをもち、型分類（A/B/C）とA型の亜型（H1–H16、N1–N9）がある。
PDF p.84-85

変異には抗原ドリフト（連続変異）と、A型で起こる抗原シフト（不連続変異）がある。
PDF p.87

臨床として、潜伏期1–3日で急激に高熱・全身症状が出て、約1週間で軽快する。
PDF p.102-103

合併症として肺炎（原発性ウイルス肺炎、混合性、二次性細菌性）や、脳症、小児の熱性けいれん、心筋炎などがある。
PDF p.104,108

治療は一般療法（休養・水分）＋抗インフルエンザ薬（オセルタミビル、ザナミビル、ラニナビル、ペラミビル、バロキサビル）を用い、早期投与が重要。
PDF p.110-113

予防は手洗い、マスク、換気、生活習慣、ワクチン。
PDF p.114

ワクチン株はシーズンごとに更新され、2025年度から3価へ。
PDF p.117

新規ワクチンとして経鼻弱毒生ワクチン（フルミスト®）、高用量ワクチン（エフルエルダ®）、mRNAワクチン、インフルエンザ・COVID混合ワクチンなどが話題。
PDF p.119-126

●COVID-19
COVID-19は2019年末から世界的流行となり、WHO緊急事態終了後も流行波を繰り返す。感染様式は飛沫、エアロゾル、接触。
PDF p.127,130

重症化リスクは年齢が大きく、基礎疾患も関与する。日本の重症度分類ではSpO2などで軽症〜重症を分類。
PDF p.134-137

治療薬は承認薬があるが適応やコスト等の問題もある。
PDF p.140

Long COVID（罹患後症状）は倦怠感、咳、集中力低下など多彩で、病因仮説として免疫異常、腸内細菌叢、自己免疫、凝固・内皮異常、神経シグナル異常などが提唱される。
PDF p.141-142

ワクチンは発症・入院・死亡を減らす効果が示され、追加接種の対象・頻度は年齢やリスクで調整される。
PDF p.145-146

●RSウイルス
RSウイルスは乳幼児で初回感染が重症化しやすいが、成人・高齢者でもリスク群では問題となり、ワクチン接種対象（高齢者やリスクの高い50歳以上など）が整理されている。
PDF p.151-152

結語：治療から予防へ

高齢者肺炎は誤嚥・ADL低下・再発の問題が大きく、発症後の機能低下や認知低下も含めて「治す」だけでなく「予防」に重点を置くべきである。講義の締めとして「肺炎は老人の友」から「ワクチンは老人の友」へ発想を転換し、ワクチンと日常対策で肺炎を減らす重要性が強調された。
PDF p.12「肺炎は老人の友である」／PDF p.153「ワクチンは老人の友である」
